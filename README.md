sparkdata
=
整理了实习半年以来的个人工作代码以及学习到的知识点

---
 * crawler: 解析爬虫组爬到的数据到hdfs中
 * FeatureRobot: 特征机器人相关代码,包括特征交叉,筛选,GBDT模型造新特征
 * fraud_item: 电商,社交异常数据筛选,包括使用规则和模型筛选
 * taobao_ciyun: 电商用户购买商品词云数据更新,包括分词,词语清洗,tfidf权重计算
